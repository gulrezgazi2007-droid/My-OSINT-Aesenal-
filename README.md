# 🕵️‍♂️ My OSINT & Security Arsenal

Welcome to my personal collection of tools and resources that I use for exploration and continuous learning! 🚀

## 🌐 Open Source Intelligence (OSINT)
*   **[Maigret](https://github.com/soxoj/maigret)** - An incredible tool for hunting down a specific username across thousands of websites.
*   **[Maltego](https://www.maltego.com/)** - My go-to application for visually mapping data and analyzing relationships.
*   **[Osintgram](https://github.com/Datalux/Osintgram)** - An excellent script for gathering public footprint information from Instagram accounts.
*   **[Sherlock](https://github.com/sherlock-project/sherlock)** - A very popular and fast tool to find usernames across social networks.
*   **[Shodan](https://www.shodan.io/)** - The search engine for Internet-connected devices. Great for discovering exposed assets.

## 🐧 Linux & Networking Basics
*   **[Nmap](https://nmap.org/)** - Essential for network discovery, port scanning, and basic security auditing.
*   **[Wireshark](https://www.wireshark.org/)** - Perfect for analyzing network traffic and understanding packet-level data.
*   **[Netcat](https://sectools.org/tool/netcat/)** - The "Swiss Army knife" of networking, useful for reading and writing data across network connections.

## 🔒 Web Security & Exploration
*   **[Burp Suite](https://portswigger.net/burp/communitydownload)** - The standard tool for hands-on web application security testing and intercepting traffic.
*   **[Have I Been Pwned](https://haveibeenpwned.com/)** - A quick web resource to check if an email or phone number has been compromised in a data breach.
